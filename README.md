dechat_i2p_prototype
====================

Минимальный рабочий прототип децентрализованного мессенджера через I2P + DHT (aiodht).
Всё хранится в RAM (локальный кэш с адаптивным TTL).

Включено:
- автопубликация в DHT при старте и периодическая републикация
- поиск по имени (точный и простая эвристика для префикса)
- RAM-only кэш с адаптивным TTL (увеличивается при обращении)
- выбор из списка совпадений при нескольких результатах
- безопасный канал: RSA обмен -> AES (Fernet) для сообщений
- I2P транспорт (через i2plib) — работает при запущенном локальном I2P роутере

Зависимости:
  pip install -r requirements.txt

Запуск:
  1) Убедись, что локальный I2P-роутер запущен (I2P Desktop / I2P Android).
  2) pip install -r requirements.txt
  3) python main.py

Примечание:
- Это прототип. Для продакшна нужны дополнительные проверки, хранение ключей между запусками (опционально), обработка ошибок сети.
